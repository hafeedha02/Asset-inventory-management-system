# Asset Inventory Management System

A full-stack Asset Inventory Management System developed using **Django REST Framework**, **React**, **PostgreSQL**, and **Docker**. The application helps organizations efficiently manage assets, assign them to employees, monitor inventory, and maintain asset records through a secure and user-friendly interface.

---

## Features

- User Authentication & Authorization
- Dashboard with asset overview
- Asset Management (Create, Update, Delete, View)
- Employee Management
- Asset Assignment & Return
- Inventory Tracking
- Search and Filter Assets
- RESTful API using Django REST Framework
- Responsive React Frontend
- PostgreSQL Database
- Dockerized Deployment

## Installation

### Clone the repository

```bash
git clone https://github.com/hafeedha02/Asset-inventory-management-system.git
```

### Navigate into the project

```bash
cd Asset-inventory-management-system
```

### Install backend dependencies

```bash
pip install -r requirements.txt
```

### Run database migrations

```bash
python manage.py migrate
```

### Start the backend server

```bash
python manage.py runserver
```

### Start the React frontend

```bash
cd asset-management-frontend
npm install
npm start
```
